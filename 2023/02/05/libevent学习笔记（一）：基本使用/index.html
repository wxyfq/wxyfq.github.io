libevent的安装比较简单，在linux上可以直接从源进行安装，需要注意的是，用于开发时应该安装dev版本，如在我的ubuntu环境上可以使用命令apt install libenvent-dev进行安装;macOS上可以使用brew install libevent进行安装。也可以从官网下载源码自行编译安装，自行编译时需要自己修改PKG_CONFIG_PATH环境变量以确保pkg-config命令能够搜索到库的安装位置；手写makefile文件的大神可以忽略上述文字。

实现的功能很简单，从标准输入读取输入，然后将读取到的数据写到标准输出。从这个简单的程序我们可以了解libevent的使用方法，分为以下几个步骤：

  1. 创建event_base;
  2. 创建event；
  3. 将创建的event添加到event_base;
  4. 开启事件循环

event_base负责保存注册的事件并且通过轮询判断哪些事件被激活。通过event_new()函数创建的事件被添加到event_base当中后，随着event_base_dispatch()函数的调用，event_base开始轮询，当某一事件被激活后，该事件的回调函数就会被调用，调用返回后轮询会继续进行。

使用上述例子中展示的四个函数，已经可以实现很多需求，并且可以代替select、poll、epoll等函数实现更好的跨平台特性，由于libevent的底层实现就是select、poll这些平台相关的api，所以性能方面也不会受到很大的影响。
